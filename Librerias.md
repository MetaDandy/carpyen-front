# Librerías del Proyecto

## 1. React
- **Descripción**: Biblioteca para construir interfaces de usuario.
- **Uso**: Se utiliza para crear componentes reutilizables y gestionar el estado de la aplicación.

## 2. React Router
- **Descripción**: Biblioteca para manejar la navegación en aplicaciones de React.
- **Uso**: Permite definir rutas y navegar entre diferentes vistas de la aplicación.

## 3. Axios
- **Descripción**: Cliente HTTP basado en promesas para hacer solicitudes a APIs.
- **Uso**: Se utiliza para realizar peticiones a servicios web y manejar respuestas.

## 4. Redux
- **Descripción**: Biblioteca para gestionar el estado global de la aplicación.
- **Uso**: Facilita el manejo del estado compartido entre componentes.

## 5. Vite
- **Descripción**: Herramienta de construcción y desarrollo para aplicaciones modernas.
- **Uso**: Proporciona un entorno de desarrollo rápido y optimiza la construcción para producción.

## 6. TypeScript
- **Descripción**: Superset de JavaScript que añade tipado estático.
- **Uso**: Mejora la calidad del código y facilita el desarrollo con autocompletado y verificación de tipos.

## 7. ESLint
- **Descripción**: Herramienta para identificar y reportar patrones en el código JavaScript.
- **Uso**: Ayuda a mantener un código limpio y consistente.

## 8. Prettier
- **Descripción**: Formateador de código que asegura un estilo consistente.
- **Uso**: Se utiliza para formatear automáticamente el código al guardar.

## 9. Tailwind CSS
- **Descripción**: Framework CSS para crear interfaces personalizadas.
- **Uso**: Proporciona clases utilitarias para un diseño rápido y responsivo.

## 10. Jest
- **Descripción**: Framework de pruebas para JavaScript.
- **Uso**: Se utiliza para realizar pruebas unitarias y de integración en la aplicación.
